# `@next/codemod` とは？

`@next/codemod` は、Next.js公式が提供している「コード自動書き換えツール」です。

Next.jsのバージョンが新しくなると、今までのコードの書き方ではエラーが出てしまうことがあります。このツールは、プロジェクト内のファイルを自動でスキャンし、新しいバージョンの仕様（ルール）に合わせてコードを自動で修正してくれます。

* **手間の削減:** 数十〜数千のファイルを一瞬で自動修正します。
* **安全性の確保:** 公式のルールに基づくため、手作業によるミスを防ぎます。
* **手動確認も必要:** ツールが自動で修正しきれない複雑な箇所は、エラーメッセージなどを見て手動で直す必要があります。

`@canary` は、このツールの「最新版（テスト版も含む）」を指定するための目印です。

## Next.js 16 への詳細なアップデート手順

Next.js 16では、`middleware` の名称変更や非同期処理の厳密化など、重要な変更がいくつか含まれています。npm環境での以下の手順に沿ってアップデートを進めてください。

### 1. 事前準備（バックアップ）
大量のコードが自動で書き換えられるため、後からやり直せるように必ず現在の状態を保存します。ターミナルで以下のコマンドを実行してください。

```bash
git add .
git commit -m "Next.js 16 アップデート前のバックアップ"
